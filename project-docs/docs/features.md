# Features

## Seiten

### Startseite
- Hero-Bereich mit Begrüßung
- Aktuelle News (automatisch die neuesten 3)
- Kommende Termine (automatisch die nächsten 3)

### Aktuelles
- Alle News-Artikel
- Kategorien (Politik, Veranstaltungen, etc.)
- Einzelansicht mit vollem Text und Bildern

### Termine
- Alle Veranstaltungen
- Status: geplant, abgesagt, vergangen
- Ort, Adresse, Uhrzeit
- Einzelansicht mit Details

### Vorstand
- Alle Vorstandsmitglieder mit Foto
- Name, Amt, Kontaktdaten

### Fraktionen
- Gemeinderatsfraktion
- Kreistagsfraktion
- Mitglieder mit Funktion

### Berichte
- Berichte aus Gemeinderat und Kreistag
- Nach Gremium filterbar
- Zusammenfassungen der Sitzungen

### Kontakt
- Kontaktformular
- Adresse, Telefon, E-Mail

### Impressum
- Rechtliche Angaben
- Datenschutz

## Interne Dokumentation (optional)

Der `docs/` Ordner kann für interne Zwecke genutzt werden:

- Geschützter Bereich (mit Cloudflare Access)
- Briefvorlagen als PDF
- Interne Anleitungen
- Kandidatenlisten
- Strategiepapiere

## Briefvorlagen

- Typst-basierte Vorlagen im CDU Corporate Design
- DIN 5008 konform
- Automatische PDF-Generierung
- Per Claude neue Briefe erstellen lassen
