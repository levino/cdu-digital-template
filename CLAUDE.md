# CDU Musterstadt Template

Website und Dokumentation für den CDU Gemeindeverband Musterstadt.

## Struktur

- `website/` - Öffentliche Website (Astro)
- `docs/` - Interne Dokumentation (Astro + Shipyard)
- `briefvorlagen/` - Typst-Vorlagen für Briefe

## Agents

- `setup` - Richtet das Template interaktiv für einen neuen Verband ein
- `briefeschreiber` - Erstellt offizielle Briefe im CDU Design
