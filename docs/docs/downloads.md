# Downloads

Hier finden Sie alle offiziellen Vorlagen und Dokumente des Gemeindeverbands.

## Briefvorlagen

Professionelle Briefvorlagen im CDU Corporate Design, DIN 5008 konform.

| Vorlage | Beschreibung | Download |
|---------|--------------|----------|
| Beispiel-Brief | Standardbrief für allgemeine Korrespondenz | [PDF herunterladen](/downloads/beispiel-brief.pdf) |

:::note[Hinweis]
Die PDF-Dateien werden automatisch aus den Typst-Vorlagen generiert. Bei Änderungen an den Vorlagen werden die PDFs beim nächsten Build aktualisiert.
:::

## Verwendung

1. Laden Sie die gewünschte Vorlage herunter
2. Öffnen Sie die PDF-Datei
3. Drucken Sie auf Ihrem Briefpapier oder verwenden Sie die Vorlage digital

## Eigene Briefe erstellen

Für individuelle Briefe nutzen Sie [Claude Code](https://claude.ai/code):

> "Erstelle einen Brief an den Bürgermeister zum Thema Straßensanierung"

> "Schreibe eine Einladung zur Jahreshauptversammlung für alle Mitglieder"

Claude verwendet automatisch die korrekten Vorlagen und das CDU Corporate Design.
